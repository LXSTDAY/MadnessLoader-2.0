# MadnessLoader-2.0
This is some like mobile plugin loader

# Установка
1. Скачиваем и устанавливаем VS Code 
2. Устанавливаем плагин C/C++
3. Качаем этот репозиторий и размещаем в папку C:/Android/ (если нету папки создаем)
4. Качаем с оф сайта NDK 16b
5. Распаковываем в ту же папку и переименовываем в "NDK-16"
6. Переходим на компьютере в Настройки >> Система >> О программе >> Дополнительные параметры системы
7. Затем Переменный среды >> переходим в Path >> создать >> вставляем строчку C:\Android\NDK-16
8. Желательно иметь на компьютере полного администратора
9. Открываем папку где распакован наш репозиторий через VS Code
10. Нажимаем "Ctrl + `" и в появившуюся консоль пишем ndk-build. Должна пойти сборка проекта
11. Переходим с помощью левого окна в папку jni/plugins/
12. Копируем папку template-plugin в эту же директорию и переименовываем как хотите назвать плагин
13. Переходим в папку вашего плагина и открываем файл Android.mk и находим строку
```make
LOCAL_MODULE := TEMPLATE
```
14. Меняем TEMPLATE на ваше название плагина
15. пробуем скомпилировать командой в консоли ndk-build ..
16. если прошло успешно в проводнике в папке репозитория создалась папка libs/armeabi-v7a/ в ней лежит будущий плагин libНАЗВАНИЕВАШЕГОПЛАГИНА.so
17. убираем из названия lib и .so меняем на .pm
18. перетаскиваем в папку игры /Android/data/com.madness.game/plugins/
19. пробуем запустить игру

# Дополнительно
1. Что бы создать новый плагин повторяем пункты с 12
2. Предварительно нужно установить апк ML и com.rockstargames.gtasa кеш полностью распакованный и переименованный в com.madness.game
3. Данный репозиторий сыроват, ждите обновлений
4. Игра не поддерживает obb кеш
